HTML (HyperText Markup Language)

HTML adalah bahasa markup yang digunakan untuk membuat struktur dan konten halaman web.

CSS (Cascading Style Sheets)

CSS adalah bahasa yang digunakan untuk mendesain dan mengatur tata letak halaman web yang telah dibuat denganHTML.

PHP (Hypertext Preprocessor)

PHP adalah bahasa skrip sisi server yang digunakan untuk menghasilkan konten web dinamis dan berinteraksi dengan basis data.
